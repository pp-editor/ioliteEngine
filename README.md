# ioliteEngine

## OverView
描画系プログラム習得のためゲームエンジンをDirectX11で作成。
安全/速い処理対応よりも機能実装を優先。

## Environment
- Windows 10
- Visual Studio 20117
- DirectX11
- nVidia GTX1080

## Implementation
- Texture Mapping
- Multiple Render Objects
- Phong-Shading
- Render CubeTexture
- Texture Class
- Shadow Mapping
- Mouse / Keyboard
