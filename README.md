# ioliteEngine

## OverView
描画系プログラム習得のためゲームエンジンをDirectX11で作成。
安全な処理対応よりも機能実装を優先。

## Implementation
- Texture Mapping
- Multiple Render Objects
- Phong-Shading
- Render CubeTexture
- Texture Class
- Shadow Mapping
